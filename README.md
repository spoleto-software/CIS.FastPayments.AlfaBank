# Spoleto.FastPayments.AlfaBank

C# SDK для интеграции с хостом Альфа-Банка для оплаты покупок через СБП (Систему Быстрых Платежей).

The project for integration with Fast Payment System based on the AlfaBank host.

## Spoleto.PaymentCallback.Service

Web API сервис для получения уведомлений. Поддержка уведомлений от Альфа-Банка и АТОЛ Онлайн.

Web API callback service for Alfa-Bank Fast payment system and Atol Online.

Работает на NET 6.
